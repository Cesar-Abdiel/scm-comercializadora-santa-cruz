# Changelog

Todos los cambios importantes de este proyecto serán documentados en este archivo.

El formato está basado en Keep a Changelog y el versionamiento sigue SemVer 2.0.0.

## [1.1.1] - 2026-07-01

### Fixed

- Corrección del cálculo de total con descuento en el módulo de facturación.
- Validación del cálculo de importes comerciales.
- Ajuste menor para asegurar resultados correctos en pedidos facturados.

## [1.1.0] - 2026-07-01

### Added

- CRUD básico de clientes.
- CRUD básico de productos.
- Registro de pedidos y detalle.
- Documentación SCM del inventario de ítems de configuración.
- Workflow inicial de GitHub Actions para build, test, lint y deploy-staging.

## [1.0.0] - 2026-07-01

### Added

- Estructura base del repositorio.
- Archivo README.md inicial.
- Carpetas principales del proyecto: docs, src, tests, config y database.
- Script inicial de base de datos.
- Línea base inicial del proyecto SCM.
